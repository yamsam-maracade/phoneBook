# ADDRESS BOOK & PLACES VISITED

## OVERVIEW
- This is a simple address book that allows users to store contact info and record places they have visited.

## FEATURES
- Add contact
- Add places

## TECHNOLOGIES USED
- HTML
- CSS
- JAVASCRIPT

## USAGE
- Open the file in your browser.
- Add contact by filling in the first name, last name and phone number.
- Fill in location, landmarks, season and add some notes to add a place.
